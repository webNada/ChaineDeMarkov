# Chaine de Markov

Projet JavaScript illustrant le fonctionnement d’une chaîne de Markov à travers une application web interactive.

## Description
Ce projet met en œuvre une chaîne de Markov afin de générer des contenus de manière probabiliste.
Il est structuré autour d’un moteur de calcul (MarkovEngine) et de pages dédiées à l’interaction utilisateur.

Le projet a été développé en JavaScript et utilise Vite comme environnement de développement.

## Fonctionnement
- Le moteur de Markov gère les états et les transitions probabilistes
- Les pages permettent d’interagir avec le modèle
- L’interface est accessible via le navigateur

## Installation et exécution

### Prérequis
- Node.js installé sur la machine

### Lancer le projet en local
cd chaine-De-Markov
npm install
npm run dev
npx vite
